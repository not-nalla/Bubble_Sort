# Bubble Sort Program in C (Numbers and Strings)

This C program demonstrates the Bubble Sort algorithm for sorting both integers and strings.

## Features
- Sorts numbers in ascending order
- Sorts strings in alphabetical order
- Uses arrays, loops, and `string.h`
- Displays data before and after sorting

## Purpose
This program is designed for learning and understanding the Bubble Sort algorithm.
